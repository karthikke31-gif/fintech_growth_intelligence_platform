# 🚀 Fintech Growth Intelligence Platform

An end-to-end FinTech analytics project designed to analyze customer growth, product adoption, payment performance, user retention, and business KPIs using **SQL, Python, PostgreSQL, Power BI, and Machine Learning**.

The project simulates a real-world fintech data warehouse and demonstrates how product, growth, and business teams can make data-driven decisions.

---

# 📌 Project Overview

This project covers the complete analytics lifecycle:

- Data Modeling
- SQL Analytics
- Exploratory Data Analysis (EDA)
- Product Metrics
- Funnel Analytics
- Cohort Analysis
- A/B Testing
- Customer Churn Prediction
- Executive KPI Reporting

---

# 🎯 Business Problem

FinTech companies continuously monitor customer engagement, transaction performance, and revenue growth.

This project answers questions such as:

- How many users are active?
- What is the platform GMV?
- How much revenue is generated?
- Which payment methods perform best?
- Where do users drop in the product funnel?
- What is the customer retention rate?
- Which customers are likely to churn?
- Which experiment performs better?
